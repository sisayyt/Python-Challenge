Summary

This project aims to answer "what is the weather like when getting close to the equator?” Python CiitiPy library, Python, API’s, and JSON tools are used to answer this question.

A random of 1500 Latitude and Longitudes was used to find the closest Lat/Long the quator to get a listing of over 600 cities. Using Open Weather API, weather data was collected from OpenWeatherMap. A dataframe was created using the data gathered from 1500 cities.

The dataframe is analyzed across the three climate zones. The relationship between latitude and temperature, latitude and humidity, latitude and cloudiness, and latitude and wind speed were plotted using MatplotLib scatter plots to view.



  • Temperature (F) vs. Latitude
  • Humidity (%) vs. Latitude
  • Cloudiness (%) vs. Latitude
  • Wind Speed (mph) vs. Latitude
